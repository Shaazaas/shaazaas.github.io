# Scary Portfolio

A minimal, spooky portfolio website with pure black background and white monospace text.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Customization

Edit `src/App.jsx` to update:
- Your name
- Work description
- About text
- Social media links
- CV/resume download link
- Portfolio link

## Features

- Pure black background (#000000)
- White text (#ffffff)
- Monospace coding font
- Minimal, spooky aesthetic
- Responsive design
